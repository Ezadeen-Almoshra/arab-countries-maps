## 1. Installation

**1. Clone the repository:**

```bash
git clone https://github.com/Ezadeen-Almoshra/arab-countries-maps.git
```

**2. Install dependencies:**

```bash
npm install
```

**3. Start the development server:**

```bash
npm run dev
```
